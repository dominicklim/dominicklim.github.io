Ninja Commuters

Ninja Commuters is a Chrome extension that helps students plan their class schedules by displaying travel time between classes.

Ninja Commuters won 1st place and a Raspberry Pi kit at HackJam 2013, a hackathon held at UC Berkeley, open to all students.

You can check it out here.

Built with: Ninja Courses and the Google Maps JavaScript API
